Getting Started with the Baomao Programming Language

By Michael Yuhas

I. Installation

Windows

1.) In your "C:\Program Files" Directory create a folder called "Baomao".

2.) Open control panel and search for 'path'. Press 'Edit environment variables for your account'. In the window that appears, select the Path variable and press 'Edit'. Press 'New' and enter C:\Program Files\Baomao. Press okay and save changes.

3.) Copy baomao.exe from cpl\bin\x86\windows10\baomao.exe into C:\Program Files\Baomao\baomao.exe

4.) Open terminal and type 'baomao'. You should see the following appear:

II. Install and IDE

You can use your favorite text editor or IDE to create Baomao programs, but I recommend using Atom. I have created an atom package that includes syntax highlighting to make reading and writing code easier. This section details the installation process.

1.) Download and install Atom on your system. Go to https://atom.io

2.) Clone my baomao language package for Atom:

git clone https://github.com/m-yuhas/language-baomao.git

3.) Copy the language package into the .atom folder for you Atom installation. Terminal commands are listed below, but you can do this using the GUI if that is more comfortable. On Linux and MacOS:

cp -r cp -r ~/git/language-baomao/ ~/.atom/packages/language-baomao/
On Windows:
mkdir .atom\packages\language-baomao
xcopy git\language-baomao .atom\packages\language-baomao /s /e

IV. Getting User Input

Now let's make a program that get's user input. We can do this with the 输入 function. 输入 will display a string in the console and collect all user input before the return key is pressed. This data will be stored as a string. Try it with the example program below:

姓名 《 输入（“你叫什么名字？”）
输出（“你好”+姓名）
Notice also that we can use the '+' operator on two strings to concatenate them. Notice also that we can set a variable using '《'. Instead of using '=' to set a variable like in other programming language, an arrow (《) is used to show that the result of the expression on the right is being stored into the variable on the left. This helps students learning programming reinforce the fact that a variable acts as a storage container for a value.

V. Branching

We also have the ability to branch depending on a condition. See the example code below:

姓名 《 输入（“你叫什么名字？”）
如果 姓名 = “我不知道”
 输出（“你怎么不知道你的名字？“）
否则
 输出（”你好“+姓名）
结束分支
In this case 如果 asks if the following condition is true or false. If the condition after 如果 is true, the commands immediately following the line with 如果 are executed. If the condition is false, the lines after 否则 are executed. After the 结束分支 statement, every line is executed regardless of the previous condition. Thus in this example the user is asked what their name is. If they reply that they do not know, then the program chastizes them. Otherwise, it greets them.

VI. Looping

Okay, let's do something a little more mathematical; let's calculate the Fibonacci sequence. We can do this with a loop:
斐波那契数号 《 1
以前来的数号 《 1
当 斐波那契数号 小于 1000
 输出（斐波那契数号）
 暂时价值 《 斐波那契数号
 斐波那契数号 《 斐波那契数号 + 以前来的数号
 以前来的数号 《 暂时价值
结束循环
In this example when the statement after 当 evaluates to true, all the statements between 当 and 结束循环 are executed. After execution completes, the statement after 当 is evaluated again and the process continues.

VII. Recursion

You can also calculate the Fibonacci sequence using recursion. In Baomao language this is possible by defining a function and then calling it recursively. We can define a function using the keyword 函数 followed by the function name. After, 要 is used to show which arguments the function takes. See the example code below for the function 斐波那脆, which takes arguments 目前 and 以前 corresponding to the current and previous Fibonacci numbers. To indicate the end of a function, use the keyword 结束函数. The code inside the function will not be executed until the function is called. You call the function by using its name followed by the list of arguments separated by commas, contained within parentheses. Furthermore, the keyword 返回 can be used at anytime to escape execution of the function:
函数 斐波那脆 要 目前、以前
 输出（目前）
 如果 目前 大于 1000
  返回
 否则
  暂时 《 目前
  目前 《 以前 + 目前
  以前 《 暂时
  斐波那脆（目前、以前）
  返回
  结束分支
结束函数

斐波那脆（1、1）
