CPL 0.0

CPL0.0版本的单证(中文)

介绍

本文件表达CPL0.0版本的详情和使用法。此版本应该有图灵完备性，还要显示本电脑语言的潜能。您发现程序错误的话，请备案在https://github.com/m-yuhas/cpl/issues。

安装

  1. 安装Swift：
    • 在macOS安装Xcode
    • 在Linux系统跟随https://swift.org/download/#releases的指示安装
  2. 从Git复制： 

    git clone https://github.com/m-yuhas/cpl.git

  3. 编译与安装： 

    chmod +x build.sh

    ./build.sh install

    安装脚本将编译和安装程序在/usr/bin/ ，这样在Bash Shell能用。如果您更偏好安装在不同的目录就用随着的命令：

    ./build.sh install /path/to/install/directory

    为卸装用一样的脚本：

    ./build.sh uninstall

    或者：

    ./build.sh uninstall /path/to/install/directory

  4. 开始编程：

    用您最喜欢的文本编辑器写程序然后存档用UTF－8码。为执行程序：

    cpl <your script>

语言句法

在0.0版本随着的命令已经被实施：

#!/usr/bin/cpl - Bash Path

描述

好像在别的电脑语言第一个句能定义Bash目录让用户能使它档案执行。只有程序第一个句能获得本字符串。

实例

#!/usr/bin/cpl

注解 - Comment

描述

定义一句的注解。本句不要被执行。

实例

注解：这是一个注解

输出 - Print Line

描述

本命令让用户印字符串或变量或表达式的求值。印的文应该在引号内。如果变量或表达式随着井号（＃）就被求值。注意：哪像别的电脑语言，各种的引号被接受无论它们的角或者定位。

实例

输出：“你好世界 ＃2+2 ＃东西”

如果 / 否则 / 结束支 - If / Then / Else

描述

本命令使程序的执行有分支。“如果”求值一个表达式：如果表达式是真正的就那些句“如果”和“否则”之间将执行。如果没有“否则”就所有的句到“结束支”将执行。那三类的表达式能被求值是：平等（=），大于（>），与小于（<）。注意哪像别的电脑语言这里只要一个等号表达平等，不需要双个等号。

实例

如果 东西=3.1

	输出：“表达式对的”

否则

	输出：“表达式否的”

结束支

从...直到 / 结束圈 - For Loop

描述

本命令拿开始的条件然后循环一段码直到最后的条件完成。注意程序员应该自己变更循环计数变量每个循环。循环计数变量在循环外可用，不过在循环内造的变量不能用循环外循环完成的以后。

实例

从 东西=1 直到 5

	输出：“循环数：＃东西”

	东西=东西+1

结束圈

当 / 结束圈 - While Loop

描述

本命令查如果一个条件真不真正的就继续执行一段码直到条件不满足。每个变量被造在循环内不能用循环完成以后。

实例

东西=0

当 东西<10

	输出：“＃东西”

	东西=东西+1

结束圈

跳出 - Break

描述

本命令让程序员从目前的循环跳出来。如果有嵌套循环本命令从最近按在堆栈的循环跳出来。

实例

循环数=0

当 1=1

	循环数=循环数+1

	如果 循环数>100

		跳出

	结束支

结束圈

输出：“＃循环数”

Storing Variables

描述

任何字符串不包括!,@,#,$,%,^,&,*,(,),[,],{,},",+,-,\,|,/,<,=,>,/,或者;并且第一个字不是数字，可用为变量。放变量用等号（=）。右边的表达式将存在左边的变量。

实例

派=3.141592654
